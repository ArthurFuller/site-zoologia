Transições Evolutivas

  1. Cordados → Vertebrados: Crânio + coluna vertebral
  2. Agnatha → Gnathostoma: Mandíbulas + nadadeiras pares
  3. Aquático → Terrestre: Pulmões + membros + pele cornificada
  4. Anfíbios → Répteis: Ovo amniótico + pele impermeável
  5. Répteis → Aves: Penas + endotermia + voo
  6. Répteis → Mamíferos: Endotermia + pêlos + glândulas mamárias
